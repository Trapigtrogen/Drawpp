◆ ellipseMode()

void DGraphics::ellipseMode ( PosMode  m)

Set the ellipse mode to m.

When ellipse mode is CENTRE, x and y will be in the center of the ellipse.
When ellipse mode is CORNER, x and y will be in the top-left corner of the rectangle around the ellipse.
Default mode is CENTRE.
